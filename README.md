# Proyecto Gestor de Gastos Personales

Aplicación web para registrar, consultar, editar y eliminar gastos personales.

## Tecnologías utilizadas

- React
- Node.js
- Express
- Supabase/PostgreSQL
- SonarQube Cloud


## Funcionalidades

- Registro de gastos
- Listado de gastos
- Edición de gastos
- Eliminación de gastos
- Cálculo del total gastado
- Conteo de registros
- Clasificación por categorías

## Arquitectura

El frontend está desarrollado con React y consume una API REST creada con Node.js y Express.  
El backend se conecta a Supabase, que funciona como base de datos PostgreSQL en la nube.

Flujo general:

React → Express → Supabase/PostgreSQL
